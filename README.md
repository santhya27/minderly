**Minderly** is a cross-platform application built with **Flutter** designed to ensure seniors never miss their medications. By connecting caretakers and seniors through a real-time cloud database, it provides a persistent voice-alarm system that only silences once the medication is confirmed as taken.

* **Frontend:** Developed using **Flutter** for a responsive, dark-themed user interface.
* **Real-Time Sync:** Uses **Firebase Firestore** to bridge the gap between Caretaker and Senior devices instantly.
* **Persistent Voice Reminders:** Utilizes Text-to-Speech (TTS) to call out medication names until the "DONE" button is pressed.
* **Live Status Tracking:** Caretakers can see a "Pending" vs. "Taken" status in real-time.
* **Live Dashboard Clock:** A digital clock on the senior interface to monitor scheduled times.
